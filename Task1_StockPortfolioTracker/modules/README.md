# Stock Portfolio Tracker

## Description

A Python-based Stock Portfolio Tracker that allows users to:

- Add stock symbols
- Enter stock quantities
- Calculate investment value
- Generate CSV reports
- Generate TXT reports

## Technologies Used

- Python
- Dictionary
- Functions
- File Handling
- CSV Module
